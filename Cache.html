type stat_info = {
   ctime : time;
   mtime : time;
   dev : Int32.t;
   inode : Int32.t;
   mode : mode; (*binary integer containg the lower 32 bits of the entry (file or symbolic link) file system entity type and permissions.*)
   uid : Int32.t;
   gid : Int32.t;
   size : Int32.t; (*binary integer containg the lower 32 bits of the entry (file or symbolic link) size.*)
}
These fields are used as a part of a heuristic to determine if the file system entity associated with this entry has changed. The names are very *nix centric but the exact contents of each field have no meaning to Git, besides exact match, except for the mode and size fields.
